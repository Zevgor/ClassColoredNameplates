# Enemy Class Plates

A simple World of Warcraft addon for TBC Classic Anniversary that colors enemy player nameplate health bars by class.

## Features
- Colors enemy player nameplates using the standard class colors
- Works with the default unit frame system

## Installation
1. Copy the addon folder into your World of Warcraft AddOns directory.
2. Reload your UI or restart the game.

## Notes
This addon works for enemy players only and uses the built-in class color data from the game.
