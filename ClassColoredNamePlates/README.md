# Class Colored Nameplates

A simple World of Warcraft addon that colors player nameplate health bars by class.

## Features
- Colors enemy player nameplates using the standard class colors
- Colors friendly player nameplates using the standard class colors
- Offers an addon options panel to enable or disable enemy and friendly coloring independently
- Note: When selecting an option, you may need to retoggle the nameplates to display properly.
- Works with the default compact unit frame system


## Installation
1. Copy the addon folder into your World of Warcraft AddOns directory.
2. Reload your UI or restart the game.
3. Open the AddOns options panel to toggle enemy and friendly nameplates.

## Notes
This addon uses the built-in class color data from the game and can be configured for enemy and friendly players separately.
